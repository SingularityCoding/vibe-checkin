# Vibe学习记 UI Design

## 1. 设计方向

### 有生命力的学习轨迹

Vibe学习记是一本文字与颜色都很克制的学习手帐：记录时专注，回看时明快。它不使用“完成任务”的压力语言，也不把连续学习包装成游戏成就。

- 页面以暖象牙白留出呼吸感。
- 珊瑚色是所有主要行动的唯一颜色。
- 芒果黄、草绿、莓粉用于学习轨迹、数据层次和内容标识。
- 浓咖啡色承载核心统计；不使用深蓝、蓝紫渐变或 AI 感霓虹色。
- 一张卡片只使用一个主强调色，正文、表单和导航保持安静。

现有的深蓝＋蓝紫头像不属于本方案。正式发布前应替换为暖色头像：暖白书页、浓咖啡描边、珊瑚轨迹、芒果黄书签，以及一枚草绿色节点。

## 2. 主题与颜色

小程序默认跟随系统的浅色／深色主题；不提供手动主题开关。深色模式保持“深色纸张”的暖感，而不是纯黑界面。

颜色规则：

- 珊瑚色是唯一主行动色；保存、记录、确认等动作不变色。
- 黄、绿、粉不承载危险操作，不替代主按钮。
- 高亮底色上的文字统一使用深色文字。
- 同步状态始终使用绿；删除状态始终使用红；颜色不单独表达含义，必须配合文字或图标。

## 3. 版式与字体

### 字体

使用系统字体，不额外加载字体文件。

| 层级 | 设计尺寸 | 使用位置 |
| --- | --- | --- |
| 页面标题 | 24px / 32px，700 | 今天、学习日志、学习统计 |
| 区块标题 | 17px / 24px，600 | 今日学习、最近 7 天、学习主题 |
| 核心数字 | 28px / 34px，700 | 连续天数、分钟数、累计值 |
| 正文 | 15px / 22px，400 | 记录内容、表单正文 |
| 元信息 | 12px / 18px，400 | 时间、标签数量、同步状态 |

### 间距与形状

| Token | 值 | 用途 |
| --- | --- | --- |
| `--vc-space-1` | `8rpx` | 图标与文字的最小间隔 |
| `--vc-space-2` | `16rpx` | 紧凑元素间距 |
| `--vc-space-3` | `24rpx` | 行内／卡片内常用间距 |
| `--vc-space-4` | `32rpx` | 页面水平边距、卡片内边距 |
| `--vc-space-5` | `40rpx` | 区块间距 |
| `--vc-space-6` | `48rpx` | 大区块间距 |
| `--vc-radius-sm` | `16rpx` | Tag、紧凑按钮 |
| `--vc-radius-md` | `24rpx` | 输入框、常规卡片 |
| `--vc-radius-lg` | `32rpx` | 概览、空状态卡片 |

- 页面左右留白固定为 `32rpx`。
- 卡片采用 `1rpx` 边框，不使用明显投影。
- 主按钮、输入控件和列表行的最小点击热区为 `88rpx × 88rpx`。
- 固定底部区域为安全区留白。

## 4. 导航

### 顶部导航

顶部保持微信原生导航栏：Tab 页的标题为 `Vibe学习记`，内容区显示“今天”“学习日志”“学习统计”；子页面直接显示“记录学习”“学习详情”“设置”等页面标题。Today 页的设置入口位于内容区标题行右侧。

### 底部导航

底部使用 TDesign Custom Tab Bar，固定展示三项：

| Tab | 图标意象 | 选中状态 |
| --- | --- | --- |
| 今天 | 书本上的对勾 | 珊瑚色图标与文字，图标外有浅珊瑚圆角底 |
| 学习日志 | 纸张／清单 | 珊瑚色图标与文字，图标外有浅珊瑚圆角底 |
| 学习统计 | 简洁柱形图 | 珊瑚色图标与文字，图标外有浅珊瑚圆角底 |

未选中项使用暖灰色图标与文字。导航区使用 `--vc-color-surface`、顶部细边框和安全区留白；不做漂浮胶囊、悬浮阴影或中间凸起按钮。

## 5. 通用界面元素

| 元素 | 视觉规则 |
| --- | --- |
| 主按钮 | 珊瑚色填充、深色文字、圆角 `--vc-radius-md`；全宽主要行动置于内容区底部 |
| 次按钮 | 透明或暖白底、珊瑚色描边与文字；不与主按钮竞争 |
| 危险按钮 | 细红描边或浅红底；仅用于删除确认页与危险区 |
| 常规卡片 | 暖白底、细边框、`--vc-radius-lg`、`32rpx` 内边距 |
| 浓咖啡统计卡 | 深咖啡底、暖白文字；只用于 Today 概览和统计连续学习卡 |
| 输入字段 | 字段标签在上，输入控件在中，帮助或错误文字在下；不把字段错误只放进 Toast |
| Tag | 默认暖白描边；选中为浅珊瑚底；数据 Tag 可使用黄、绿、粉的浅色底，但文字始终深色 |
| 空状态 | 轻量书本／轨迹图标、一句标题、一句说明、一个明确行动按钮 |
| 同步提示 | 绿色图标加文字；失败使用浅红提示卡与重试行动 |

界面使用 TDesign 的按钮、输入、多行输入、步进器、Tag、Cell、图标、加载、确认框和自定义 Tab Bar。日历、趋势图和记录卡使用本产品自己的视觉样式。

## 6. 页面设计

### Today

```text
今天                                                [设置]
周三 · 7 月 15 日

┌ 浓咖啡色「学习概览」卡 ─────────────────────────┐
│ 当前连续        今日投入        今日记录          │
│ 3 天            85 分钟         2 条             │
│                       · 珊瑚 · 黄 · 绿轨迹装饰     │
└────────────────────────────────────────────────┘

最近 7 天                         7 月 09 日 — 15 日
[ 三 ] [ 四 ] [ 五 ] [ 六 ] [ 日 ] [ 一 ] [ 今 ]
   ·     ·           ·                 ●

今日学习
┌ 暖白行动卡 ────────────────────────────────────┐
│ 今天已经开始学习了                              │
│ 已记录 2 次，继续留下下一段学习轨迹。            │
│                              [再记录一次学习]   │
└────────────────────────────────────────────────┘

今日记录
┌ 20:40 · 45 分钟     [Agent] [MCP] ────────────┐
│ 给学习助手补完了工具调用的边界……                │
└────────────────────────────────────────────────┘
```

- 概览卡是唯一的大面积深色视觉；没有记录时显示真实的 `0 天 / 0 分钟 / 0 条`，不制造奖励或假轨迹。
- 近 7 天以小圆点或短线标记有记录的日期，不使用火焰图标。
- 首次使用的行动卡标题为“今天还没有开始学习”，主按钮为“记录第一次学习”。
- 今日记录卡整张可点击；内容最多两行，其余以省略号截断。

### 学习日志

```text
[近 7 天日期条：未选中为描边，选中为浅珊瑚底]

┌ 20:40 · 45 分钟  [Agent] [MCP] ──────────────┐
│ 给学习助手补完了工具调用的边界……               │
└──────────────────────────────────────────────┘

                                                     [＋]
```

- 日期和 Tag 为横向滚动控件；选中项同时改变文字、填充色和边框。
- 范围摘要与结果摘要使用元信息样式，不与统计卡竞争。
- 无筛选结果显示“清除全部筛选”；无记录显示“记录第一次学习”。
- 悬浮新增按钮避开底部导航与安全区，使用珊瑚色圆形底和深色加号。

### 学习统计

```text
学习统计

┌ 浓咖啡色「连续学习」卡 ─────────────────────────┐
│ 当前连续  6 天                                  │
│ 继续保持自己的学习节奏                           │
│ 最长连续  12 天                                 │
└────────────────────────────────────────────────┘

┌ 累计打卡日 18 天 ──┐  ┌ 累计投入 1,245 分钟 ──┐
└────────────────────┘  └────────────────────────┘

本月学习日历                         [日历区]
最近 7 天投入趋势                    [柱形区]
学习主题                             [Top 3 排名]
```

- 零记录时只展示空状态与“去记录第一次学习”，不展示空图表。
- 有记录后显示连续学习卡、累计卡、日历、趋势和主题排行。
- 日历有记录日使用珊瑚色圆点或圆角底色；趋势柱统一用珊瑚色，最高值可用芒果黄强调。
- 每个趋势值都有日期和分钟数文字；Top 3 使用序号、标签和记录次数，不使用奖杯图标。

### 记录编辑

```text
记录学习
今天 · 2026 年 7 月 15 日

学习内容 *
[ 这段时间学了什么？                              ]
                                      0 / 300

学习时长 *
[  −   ]              30 分钟               [  ＋  ]

学习主题
[选择或新建主题]  [Agent ×] [微信小程序 ×]

今日收获（选填）
[ 写下一个想在下次继续的想法……                     ]

─────────────────────────────────────────────────
                        [保存记录]
```

- 日期以只读提示呈现，不设计为可点击控件。
- 保存按钮固定在底部安全区上方；保存中显示“保存中”并禁用。
- 编辑状态沿用同一表单；“删除此记录”独立置于表单末尾，不与保存按钮并列。

### 学习详情

- 日期、星期、创建时间与学习时长置于顶部元信息区。
- 标签紧随元信息；没有标签时不留空位。
- 学习内容与今日收获为完整文本区块，保留换行；没有收获时不显示该区块。
- 右上角仅有“编辑”，不在详情页直接放删除操作。

### 设置

设置使用四个暖白 Cell Group：学习偏好、数据与同步、数据与隐私、关于。

- 同步状态置于“云端同步”条目下方，使用绿色图标与上次同步时间。
- “删除我的全部学习记录”位于最后的危险区，使用浅红底或细红描边。
- 设置页不出现头像、昵称、个性主题或主题切换入口。

## 7. 状态与反馈

| 状态 | 视觉表现 | 行动 |
| --- | --- | --- |
| 首次／零记录 | 书本／轨迹图标、标题、一句说明 | 一个主按钮，去记录学习 |
| 局部加载 | 区块内加载指示，不遮住已成功读取的内容 | 无额外行动 |
| 保存中 | 主按钮显示“保存中”并禁用 | 防止重复提交 |
| 同步成功 | 绿色图标加“已同步”或上次同步时间 | 不额外弹 Toast |
| 同步／读取失败 | 浅红提示卡，明确说明失败 | “重试”或“重新同步” |
| 表单校验失败 | 字段边框与字段下方错误文字 | 用户输入保留在原处 |
| 删除确认 | 清晰的影响说明与危险按钮 | 默认取消，确认按钮为“确认删除” |

## 8. 可访问性

- 正文与背景对比度至少为 4.5:1。
- 浅珊瑚、黄、粉等高亮底色上的文字一律使用深色。
- 图标操作配合可见文字或明确上下文，不能只依赖图形表达复杂后果。
- 长内容、Tag 和动态数字可以换行或截断，但不能重叠。
- 不依赖 hover，不把关键信息只放在长按提示中。

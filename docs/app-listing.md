# Vibe学习记 App Listing

## 产品名称

**Vibe学习记**

`Vibe` 占 4 个字符，“学习记”占 6 个字符，合计 10 个字符。仓库和内部工程名使用 `vibe-checkin`。

中文解释语：**记录每一段学习投入**

## 简介

> 轻量记录学习内容、时长与收获，回看每一段持续投入。

## 小程序介绍

> Vibe学习记是一款轻量学习投入记录工具。记录学习内容、时长、主题和收获；通过日志、连续学习与统计，回看自己的学习轨迹。

## 课程与 README 介绍

> Vibe学习记是 Vibe Coding 课程的协作实战项目。8 位同学围绕明确的 Spec、独立分支和代码评审，共同完成一个可以正式使用的微信小程序。

## 核心功能短句

- 快速记录：学习内容、时长、主题与收获。
- 学习日志：按日期、主题与关键词回找记录。
- 学习统计：看见连续学习、投入时长和主题积累。
- 云端同步：同一微信身份跨设备读取自己的记录。
